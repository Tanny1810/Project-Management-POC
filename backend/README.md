# Employee Management Backend (POC)

Backend scaffold for a FastAPI + PostgreSQL + SQLAlchemy proof of concept.

## Tech Stack
- Python 3
- FastAPI
- PostgreSQL
- SQLAlchemy
- Pydantic

## Quick Start
1. Create and activate a virtual environment.
2. Install dependencies:
   pip install -r requirements.txt
3. Copy `.env.example` to `.env` and update values.
4. Run app:
   uvicorn app.main:app --reload

## Scope
This scaffold intentionally excludes API routes and business logic.
